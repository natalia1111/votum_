✅ **СДЕЛАНО**

- Датировка голосов скрыта
- Невозможно узнать, когда был отправлен голос

## 🎯 ТВОИ ДЕЙСТВИЯ (за 15-20 минут):

1. **Создай Google Таблицу** (2 мин)
   - Две вкладки: posobie, praktikum
   - Заголовки в каждой вкладке

2. **Установи Google Apps Script** (3 мин)
   - Скопируй код из google-apps-script.js
   - Разверни как веб-приложение
   - Скопируй webhook URL

3. **Создай Telegram бота** (3 мин)
   - @BotFather → /newbot
   - Получи токен и Chat ID

4. **Заполни CONFIG в HTML** (2 мин)
   - Вставь три значения
   - Включи GOOGLE_SHEETS_ENABLED: true
   - Включи TELEGRAM_ENABLED: true

5. **Тестируй** (5 мин)
   - Открой steam_vote.html
   - Ответь на все вопросы
   - Проверь Google Sheets
   - Проверь Telegram
   - Проверь пароль 1407

6. **Готово!** ✨
   - Отправляй ссылку людям
   - Смотри результаты в реальном времени
