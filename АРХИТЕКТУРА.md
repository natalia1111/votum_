## 📍 Три хранилища одновременно:

### 1️⃣ LocalStorage (браузер)
- **Где:** На компьютере юзера
- **Зачем:** Быстрая работа, не нужен интернет
- **Безопасность:** Зависит от браузера
- **Если очистить кэш:** Данные потеряются

### 2️⃣ Google Sheets (облако)
- **Где:** На серверах Google
- **Зачем:** Резервная копия, совместный доступ
- **Безопасность:** Защищена Google аккаунтом
- **Если сломается:** Данные сохранятся

### 3️⃣ Telegram (облако)
- **Где:** На серверах Telegram
- **Зачем:** Уведомления в реальном времени
- **Безопасность:** Шифрование Telegram
- **Если сломается:** История всех сообщений остаётся

## 🔐 Безопасность и конфиденциальность

### Пароль (1407)
```
Браузер → Проверка пароля (клиент-сайд) → Доступ к статистике
```
- Пароль НЕ отправляется на сервер
- Проверка происходит в браузере
- Достаточно для защиты от случайного доступа

### Telegram Token
```
HTML → fetch() → Telegram API → Отправка сообщения
```
- Отправляется напрямую в Telegram API
- Используется HTTPS (зашифровано)
- Токен не видят другие люди (если ты его не публикуешь)

### Google Sheets Webhook
```
HTML → fetch() → Google Apps Script → Google Sheets
```
- Использует публичный webhook (но ссылка уникальна)
- Данные отправляются через HTTPS
- Только этот webhook может писать в таблицу

## 📊 Поток данных (пошагово)

```
1. ЮЗЕР ОТВЕЧАЕТ НА ВОПРОС
   ├─ Открыл steam_vote.html
   ├─ Выбрал ответы (в браузере)
   └─ Нажал "Отправить"

2. ВАЛИДАЦИЯ (на клиенте)
   ├─ Проверка: все ли вопросы ответены?
   ├─ Если нет → показать ошибку
   └─ Если да → продолжить

3. СОХРАНЕНИЕ ЛОКАЛЬНО
   ├─ Данные в localStorage браузера
   ├─ Можно отправить повторно
   └─ Уведомление: "✓ Голос сохранён"

4. ОТПРАВКА В ОБЛАКО (async - одновременно)
   │
   ├─→ Google Sheets
   │   ├─ fetch() → webhook URL
   │   ├─ Данные добавляются в таблицу
   │   └─ Задержка: 1-3 сек
   │
   └─→ Telegram
       ├─ fetch() → api.telegram.org
       ├─ Отправляется сообщение в чат
       └─ Задержка: 0.5-2 сек

5. ФОРМА ОЧИЩАЕТСЯ
   └─ Готово к следующему голосу
```

## 📈 Масштабируемость

Текущее решение выдержит:
- ✅ До 1000 голосов в день (Google Sheets)
- ✅ До 30 сообщений в секунду (Telegram)
- ✅ Неограниченное количество юзеров (браузер)

Ограничения:
- 1 webhook на таблицу (можно создать несколько таблиц)
- 1 Telegram бот (можно создать несколько ботов)
- 5 МБ на браузер (localStorage)
