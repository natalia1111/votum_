## 🔧 Краткая техническая справка

### Конфиг (нужно заполнить):
```javascript
const CONFIG = {
  GOOGLE_SHEET_WEBHOOK: 'https://...',  // ← из Google Sheets
  TELEGRAM_TOKEN: '123...',               // ← от @BotFather
  TELEGRAM_CHAT_ID: '456...',             // ← из getUpdates
  GOOGLE_SHEETS_ENABLED: true,
  TELEGRAM_ENABLED: true,
};
```

### Пароль (можно изменить):
```javascript
const CORRECT_PASSWORD = '1407';  // ← измени на свой
```

### Основные функции:
```javascript
switchPage(page)           // Переключение вкладок
submitForm(type)           // Отправка формы
renderStats()              // Показ статистики
sendToCloud(type, entry)   // Отправка в Sheets + Telegram
exportCSV()                // Экспорт CSV
exportJSON()               // Экспорт JSON
clearAll()                 // Очистка всех данных
```

## 🔒 Безопасность - что нужно знать

### Безопасно выкладывать:
- ✅ steam_vote.html (содержит только интерфейс)
- ✅ ИНСТРУКЦИИ (не содержат приватные данные)

### НЕ выкладывать:
- ❌ steam_vote.html с заполненным CONFIG
  - Если выложил → создать нового бота в @BotFather
- ❌ Токены и Chat ID в GitHub/публичных местах

### Что делать если выложил случайно:
1. Скорее всего ничего страшного (токены используются ограниченное время)
2. Но лучше создать нового бота
3. И обновить токен в CONFIG
