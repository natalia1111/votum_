# ⚡ БЫСТРЫЙ СТАРТ (5 минут)

## 🚀 Если ты в спешке:

### 1️⃣ Google Sheets (КОПИ-ПАСТА, 2 минуты)

1. Откройте https://sheets.google.com
2. **Новая таблица** → назови "STEAM Опрос"
3. В левом меню сделай **2 листа:**
   - Правый клик на листе → Дублировать → переименуй в "posobie"
   - Правый клик → Дублировать → переименуй в "praktikum"
4. На листе "posobie" в первой строке напиши заголовки:
   ```
   Рекомендация | Актуальность содержания | Полнота тем | Структура (1-5) | Навигация | Мультимедиа | Наглядность | Планируете использовать
   ```
5. На листе "praktikum" в первой строке:
   ```
   Количество заданий | Закрепление теории | Разнообразие заданий | Качество иллюстраций (1-5)
   ```

6. **Расширения → Apps Script**
7. Удали всё, вставь код из файла `google-apps-script.js` (просто Ctrl+A → Ctrl+V)
8. **Сохрани** (Ctrl+S)
9. Кнопка **«Развернуть»** (вверху справа) → **Новое развертывание**
   - Тип: **Веб-приложение**
   - Выполнять от имени: твой аккаунт
   - Имеет доступ: **Все**
10. **Развернуть**
11. **СКОПИРУЙ URL** из окна (строка "URL развертывания")
    ```
    https://script.google.com/macros/d/1AIdyR5dv4FbqLjzlN8a9K0pQ1R2S3T4U5V6W7X8/userweb?v=1
    ```

✅ **Google Sheets готова!**

### 2️⃣ Telegram Бот (КОПИ-ПАСТА, 3 минуты)

1. Открой **Telegram → @BotFather**
2. Напиши `/newbot`
3. На первый вопрос: `STEAM Опрос` (или как-нибудь)
4. На второй: `steam_vote_bot_123` (добавь в конце цифры)
5. **СКОПИРУЙ токен** (выглядит так):
   ```
   123456789:ABCdefGHIjklmnoPQRstuvWXyz1234567890
   ```

6. В том же Telegram напиши боту `/start`
7. Открой в браузере эту ссылку:
   ```
   https://api.telegram.org/bot123456789:ABCdefGHIjklmnoPQRstuvWXyz1234567890/getUpdates
   ```
   (вместо `123456789:ABCdefGHIjklmnoPQRstuvWXyz1234567890` вставь ТВОЙ токен)
8. Нажми Ctrl+F, ищи `"id":` — число рядом это **Chat ID**
   ```
   "id":987654321,
   ```

✅ **Telegram готов!**

### 3️⃣ Включи в HTML (КОПИ-ПАСТА, 30 секунд)

Открой файл `steam_vote.html` в текстовом редакторе (например, VS Code или блокнот).

Найди строку (примерно на строке 577):
```javascript
const CONFIG = {
  GOOGLE_SHEET_WEBHOOK: 'https://script.google.com/macros/d/YOUR_DEPLOYMENT_ID/userweb?v=1',
  TELEGRAM_TOKEN: 'YOUR_TELEGRAM_BOT_TOKEN',
  TELEGRAM_CHAT_ID: 'YOUR_CHAT_ID',
  GOOGLE_SHEETS_ENABLED: false,
  TELEGRAM_ENABLED: false,
};
```

**Замени на свои данные:**
```javascript
const CONFIG = {
  GOOGLE_SHEET_WEBHOOK: 'https://script.google.com/macros/d/1AIdyR5dv4FbqLjzlN8a9K0pQ1R2S3T4U5V6W7X8/userweb?v=1',
  TELEGRAM_TOKEN: '123456789:ABCdefGHIjklmnoPQRstuvWXyz1234567890',
  TELEGRAM_CHAT_ID: '987654321',
  GOOGLE_SHEETS_ENABLED: true,
  TELEGRAM_ENABLED: true,
};
```

✅ **Готово! Сохрани файл.**

## 🆘 Если не работает

**Шаг 1: Открой F12 → Console**
```
F12 → вкладка Console
```

**Шаг 2: Отправь голос и посмотри ошибки**

**Частые ошибки:**

| Ошибка | Решение |
|--------|---------|
| `Could not fetch` | Проверь URL webhook'а из Google Sheets |
| `CORS error` | Это нормально для Google Sheets (всё равно работает) |
| `TypeError: Cannot read...` | Проверь, что все три значения в CONFIG заполнены |
| Telegram не пришло | Проверь Chat ID и Token, убедись что TELEGRAM_ENABLED: true |

## 💡 Если хочешь отключить на время


Данные всё равно будут сохраняться локально (в браузере), но в облако не пойдут.
