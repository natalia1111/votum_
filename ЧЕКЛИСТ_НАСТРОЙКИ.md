## 📋 ШАГ 4: Заполнение конфига в HTML (2 минуты)

### 4.1 Открытие файла
- [ ] Открыл файл `steam_vote.html` в текстовом редакторе
- [ ] (Например, VS Code, Sublime Text, или даже Блокнот)

### 4.2 Поиск конфига
- [ ] Нажал **Ctrl+F**
- [ ] Искал: `const CONFIG = {`
- [ ] Нашел примерно на строке 577

### 4.3 Замена значений
Найди эти строки:
```javascript
const CONFIG = {
  GOOGLE_SHEET_WEBHOOK: 'https://script.google.com/macros/d/YOUR_DEPLOYMENT_ID/userweb?v=1',
  TELEGRAM_TOKEN: 'YOUR_TELEGRAM_BOT_TOKEN',
  TELEGRAM_CHAT_ID: 'YOUR_CHAT_ID',
  GOOGLE_SHEETS_ENABLED: false,
  TELEGRAM_ENABLED: false,
};
```

Замени их на:
```javascript
const CONFIG = {
  GOOGLE_SHEET_WEBHOOK: 'https://script.google.com/macros/d/[СКОПИРОВАННЫЙ_WEBHOOK_URL]',
  TELEGRAM_TOKEN: '[СКОПИРОВАННЫЙ_ТОКЕН]',
  TELEGRAM_CHAT_ID: '[СКОПИРОВАННЫЙ_CHAT_ID]',
  GOOGLE_SHEETS_ENABLED: true,
  TELEGRAM_ENABLED: true,
};
```

**Проверки:**
- [ ] Вставил полный Webhook URL (начинается с https, заканчивается на /userweb?v=1)
- [ ] Вставил полный Telegram Token (формат: цифры:буквы)
- [ ] Вставил Chat ID (только цифры)
- [ ] Изменил GOOGLE_SHEETS_ENABLED на **true**
- [ ] Изменил TELEGRAM_ENABLED на **true**
- [ ] Сохранил файл (Ctrl+S)
