# ❓ FAQ & Troubleshooting

## 🔴 Google Sheets не заполняется

### Проблема 1: "Нет данных в таблице после отправки"

**Проверь по порядку:**

1. **Webhook URL скопирован правильно?**
   - Откройте Google Apps Script
   - **Развертывание** (в левом меню) → копируйте "URL развертывания"
   - Убедись, что заканчивается на `/userweb?v=1`

2. **Листы названы правильно?**
   - Листы должны называться ТОЧНО:
     - `posobie` (маленькие буквы)
     - `praktikum` (маленькие буквы)
   - Кликните на листе → переименуй

3. **Заголовки совпадают?**
   - Убедись, что заголовки в Google Sheets совпадают с кодом:
   
   **Для листа "posobie":**
   ```
   Рекомендация | Актуальность содержания | Полнота тем | Структура (1-5) | Навигация | Мультимедиа | Наглядность | Планируете использовать
   ```
   
   **Для листа "praktikum":**
   ```
   Количество заданий | Закрепление теории | Разнообразие заданий | Качество иллюстраций (1-5)
   ```

4. **Консоль браузера показывает ошибки?**
   - F12 → Console → отправь голос
   - Скопируй ошибку

### Решение: Полная переустановка

1. Создай **новую** таблицу
2. Переустанови Apps Script (скопируй код заново)
3. Получи новый Webhook URL
4. Вставь в CONFIG

## 🔴 Telegram не приходит

### Проблема 1: "Сообщения не приходят"

**Проверь:**

1. **TELEGRAM_ENABLED: true?**
   ```javascript
   TELEGRAM_ENABLED: true,  // ← должно быть true
   ```

2. **Token верный?**
   - Должен выглядеть так: `123456789:ABCdefGHI...`
   - Попробуй открыть в браузере:
     ```
     https://api.telegram.org/botТВОЙ_ТОКЕН/getMe
     ```
   - Если видишь `"ok":true` — токен правильный

3. **Chat ID верный?**
   - Откройте в браузере:
     ```
     https://api.telegram.org/botТВОЙ_ТОКЕН/getUpdates
     ```
   - Найти `"id":123456789` — скопировать в TELEGRAM_CHAT_ID

4. **Ты написал боту /start?**
   - Откройте Telegram
   - Найди своего бота
   - Нажми /start
   - (Иначе бот не может писать)

5. **Консоль браузера показывает ошибку?**
   - F12 → Console → отправь голос
   - Посмотри ошибку

### Проблема 2: "Я не помню Chat ID"

Способ 1 (быстро):
1. Напиши боту сообщение
2. Открой в браузере:
   ```
   https://api.telegram.org/botТВОЙ_ТОКЕН/getUpdates
   ```
3. Ищи `"id":`

Способ 2 (используя @userinfobot):
1. Найди в Telegram `@userinfobot`
2. Напиши ему `/start`
3. Он пришлёт твой Chat ID

### Решение: Переинициализация бота

1. Найди своего бота в Telegram
2. Напиши `/start`
3. Попробуй отправить голос снова

## 🟡 Пароль не работает

### Проблема: "Пароль 1407 не принимает"

**Решение:**

1. **Убедись, что вводишь на английской раскладке**
   - Цифры одинаковые везде, но иногда спецсимволы отличаются
   
2. **Проверь консоль (F12)** на ошибки при вводе пароля

3. **Если ничего не помогает:**
   - Открой steam_vote.html в редакторе
   - Найди строку:
     ```javascript
     const CORRECT_PASSWORD = '1407';
     ```
   - Убедись, что пароль туда записан

## 🟡 Данные не сохраняются локально

### Проблема: "Я ответил, но данные не появились"

**Причины:**

1. **Браузер заблокировал localStorage**
   - Это может быть в приватном режиме браузера
   - Используй обычное окно браузера

2. **Переполнена память браузера**
   - Очисти кэш (Ctrl+Shift+Delete)
   - Попробуй снова

3. **Ты не полностью заполнил форму**
   - Должны быть **ВСЕ** вопросы ответены
   - Если пропустил хотя бы один — форма не отправится

### Решение:

1. Открой F12 → Application → LocalStorage
2. Ищи ключ `steam_votes`
3. Если его нет — данные не сохранены

## 🟡 Статистика показывает неправильные числа

### Проблема: "Итого не совпадает с количеством голосов"

**Причина:**

Если я ответил один раз на вопрос 1, и ответил второй раз на вопрос 2 — считается как 2 голоса, но для разных вопросов.

**Это нормально!** Статистика считает ответы по каждому вопросу отдельно.

### Проверка:

1. Открой вкладку "Статистика" (введи пароль 1407)
2. Посмотри на цифры в конце каждой карточки
3. Это количество ответов именно на этот вопрос

## 🟢 Я хочу добавить свои вопросы

### Добавить вопрос в пособие:

1. Открой `steam_vote.html` в редакторе
2. Найди секцию `<!-- Q8 -->` (последний вопрос)
3. Скопируй код одного вопроса и вставь после него:

```html
<!-- Q9 -->
<div class="q-card">
  <span class="q-num">ВОПРОС 09</span>
  <div class="q-label">ТВОЙ ВОПРОС ЗДЕСЬ</div>
  <div class="radio-group">
    <label class="radio-option">
      <input type="radio" name="p_myvariable" value="Ответ 1">
      <span class="pill">Ответ 1</span>
    </label>
    <label class="radio-option">
      <input type="radio" name="p_myvariable" value="Ответ 2">
      <span class="pill">Ответ 2</span>
    </label>
  </div>
</div>
```

4. В коде JavaScript найди где собираются поля:
```javascript
if (type === 'posobie') {
  fields = [
    // ...
    ['p_future','Планируете использовать'],  // ← последний
  ];
}
```

5. Добавь перед закрывающей скобкой:
```javascript
['p_myvariable','Мой вопрос'],
```

6. В Google Sheets добавь новый заголовок в листе "posobie"

## 🟢 Интеграция работает, но медленно

**Это нормально!**

- Google Sheets может добавлять данные с задержкой 1-3 секунды
- Telegram может доставлять уведомление с задержкой 0.5-2 секунды

Это не ошибка.

## 🟢 Я хочу экспортировать данные

2. Введи пароль 1407
3. Нажми "⬇ Экспорт CSV"

### Способ 2: JSON (для резервной копии)

1. Откройте вкладку "Статистика"
2. Введи пароль 1407
3. Нажми "⬇ Экспорт JSON"
4. Сохранится файл (можно использовать для импорта позже)
