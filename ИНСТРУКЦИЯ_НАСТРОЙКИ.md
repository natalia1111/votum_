# 📋 Инструкция по настройке интеграций

## 🔵 ЧАСТЬ 1: Google Sheets (автоматическое заполнение таблицы)

### Шаг 1: Создай Google Таблицу

1. Перейди на https://sheets.google.com
2. Создай новую таблицу (назови её "STEAM Опрос")
3. В первом листе создай два листа:
   - **Лист 1: "posobie"** — для пособия
   - **Лист 2: "praktikum"** — для практикума

4. В первой строке каждого листа добавь заголовки:
   ```
   Рекомендация | Актуальность содержания | Полнота тем | Структура (1-5) | Навигация | Мультимедиа | Наглядность | Планируете использовать
   ```
   (для "praktikum": Количество заданий | Закрепление теории | Разнообразие заданий | Качество иллюстраций (1-5))

### Шаг 2: Создай Google Apps Script

1. Открой свою таблицу
2. Нажми **Расширения → Apps Script**
3. Удали весь код и вставь этот:

```javascript
function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const sheet = SpreadsheetApp.getActiveSpreadsheet().getSheetByName(data.type);
    
    if (!sheet) {
      return ContentService.createTextOutput('Sheet not found');
    }
    
    const values = [];
    const keys = Object.keys(data.data);
    
    // Убедись, что заголовки совпадают
    for (const key of keys) {
      values.push(data.data[key]);
    }
    
    sheet.appendRow(values);
    
    return ContentService.createTextOutput('OK');
  } catch (error) {
    return ContentService.createTextOutput('Error: ' + error.message);
  }
}
```

4. Сохрани скрипт (Ctrl+S)

### Шаг 3: Разверни скрипт как веб-приложение

1. Нажми **Развернуть → Новое развертывание**
2. В выпадающем меню выбери **Веб-приложение**
3. Заполни:
   - **Выполнять от имени:** твой Google-аккаунт
   - **Имеет доступ:** Все
4. Нажми **Развернуть**
5. **СКОПИРУЙ** URL вида:
   ```
   https://script.google.com/macros/d/1ABCDEF123.../userweb?v=1
   ```

### Шаг 4: Вставь URL в HTML

Открой `steam_vote.html` и найди строку:
```javascript
GOOGLE_SHEET_WEBHOOK: 'https://script.google.com/macros/d/YOUR_DEPLOYMENT_ID/userweb?v=1',
```

Замени `YOUR_DEPLOYMENT_ID` на своё значение, получилось:
```javascript
GOOGLE_SHEET_WEBHOOK: 'https://script.google.com/macros/d/1ABCDEF123.../userweb?v=1',
```

### Шаг 5: Включи Google Sheets в конфиге

Найди:
```javascript
GOOGLE_SHEETS_ENABLED: false,
```

Измени на:
```javascript
GOOGLE_SHEETS_ENABLED: true,
```

✅ **Google Sheets готова!** Теперь все голоса будут автоматически добавляться в таблицу.

## 🔴 ЧАСТЬ 2: Telegram Бот (уведомления о новых голосах)

### Шаг 1: Создай Telegram Бота

1. Открой Telegram и найди **@BotFather**
2. Напиши `/start` → `/newbot`
3. Ответь на вопросы:
   - **Имя бота:** STEAM Опрос (или как хочешь)
   - **Username бота:** steam_vote_bot (или steam_any_unique_name)
4. **СКОПИРУЙ** токен вида:
   ```
   123456789:ABCdefGHIjklmnoPQRstuvWXyz1234567890
   ```

### Шаг 2: Найди свой Chat ID

1. Отправь боту сообщение (напиши `/start`)
2. Открой ссылку в браузере:
   ```
   https://api.telegram.org/botТВОЙ_ТОКЕН/getUpdates
   ```
   Замени `ТВОЙ_ТОКЕН` на полученный токен
3. Найди строку `"id":123456789` — это твой **Chat ID**

### Шаг 3: Вставь данные в HTML

Открой `steam_vote.html` и найди конфиг:

```javascript
TELEGRAM_TOKEN: 'YOUR_TELEGRAM_BOT_TOKEN',
TELEGRAM_CHAT_ID: 'YOUR_CHAT_ID',
```

Замени на свои значения:
```javascript
TELEGRAM_TOKEN: '123456789:ABCdefGHIjklmnoPQRstuvWXyz1234567890',
TELEGRAM_CHAT_ID: '987654321',
```

### Шаг 4: Включи Telegram в конфиге

Найди:
```javascript
TELEGRAM_ENABLED: false,
```

Измени на:
```javascript
TELEGRAM_ENABLED: true,
```

✅ **Telegram готов!** Теперь при каждом новом голосе ты получишь уведомление в бот.

## ⚠️ Возможные ошибки

### "Ошибка CORS" в консоли браузера
**Это нормально для Google Sheets!** Скрипт всё равно работает (режим `no-cors`).

### Telegram не присылает сообщения
- Проверь, что бот добавлен правильно
- Убедись, что в конфиге `TELEGRAM_ENABLED: true`
- Проверь Chat ID в Telegram
- Убедись, что токен скопирован полностью

### Google Sheets не заполняется
- Проверь, что URL скопирован правильно
- Убедись, что в конфиге `GOOGLE_SHEETS_ENABLED: true`
- Проверь, что листы называются "posobie" и "praktikum"
- Проверь наличие заголовков в листах

## 💡 Советы

### Защити свой Google Sheets
1. Открой таблицу
2. **Инструменты → Защита листа**
3. Защити лист от случайного редактирования

### Резервная копия
Регулярно скачивай данные через экспорт CSV/JSON на странице статистики.

### Отключение интеграции
Если что-то сломалось, просто измени в конфиге:
```javascript
GOOGLE_SHEETS_ENABLED: false,
TELEGRAM_ENABLED: false,
```

И всё снова будет работать локально (данные остаются в браузере).
